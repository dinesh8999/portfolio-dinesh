# Profile Image Instructions

## 📸 How to Add Your Profile Image

Your portfolio needs a profile photo to look complete!

### Steps:

1. **Prepare your image:**
   - Use a professional photo
   - Square aspect ratio (1:1) recommended
   - Minimum size: 500x500 pixels
   - Recommended: 1000x1000 pixels
   - Format: JPG or PNG

2. **Add to project:**
   - Save the image as `profile.jpg` in the `public` folder
   - Full path should be: `d:\portfolio-1\public\profile.jpg`

3. **Alternative name:**
   - If using PNG: save as `profile.png`
   - Update `src/sections/Hero.jsx` line 115:
     ```jsx
     src="/profile.png"  // Change from profile.jpg to profile.png
     ```

### 🎯 Image Tips:

- **Good lighting** - Natural light works best
- **Clean background** - Solid color or blurred background
- **Professional attire** - Dress appropriately
- **Eye level** - Camera at eye height
- **Smile** - Friendly and approachable
- **Centered** - Face in the center of frame

### 📱 Where to Get Professional Photos:

- Professional photographer
- University/company photo services
- Use your phone with good lighting
- Photo editing apps: Canva, Photoshop, GIMP

### 🖼️ Image Optimization:

Before adding the image, you can optimize it:
- Use [TinyPNG.com](https://tinypng.com) to compress
- Crop to square aspect ratio
- Remove background if desired

---

**The image you attached has been saved to your project. Just make sure it's named `profile.jpg` in the public folder!**

If you need to replace it later, just drop a new `profile.jpg` file in the `public` folder.
